Question-1: What are the different ways to select an element in the DOM?

Answer : 1. Selecting by ID Method: document.getElementById() Description: Selects a single element by its unique id attribute. Returns: A single Element object or null if no matching element is found. 2. Selecting by Class Name Method: document.getElementsByClassName() Description: Selects all elements that have a specific class name. Returns: A live HTMLCollection of elements (array-like object). 3. Selecting by Tag Name Method: document.getElementsByTagName() Description: Selects all elements with a specific tag name (e.g., div, p, a). Returns: A live HTMLCollection of elements.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Answer : 1. innerHTML: Retrieves or sets the HTML content inside an element, including all HTML tags and text. When setting, it interprets the string as HTML and renders it. 2. innerText: Retrieves or sets the visible text content of an element, excluding hidden elements (e.g., elements with display: none or visibility: hidden).It respects CSS styling and layout, so it only returns text that is actually rendered on the page 3. textContent: What it does: Retrieves or sets the text content of an element, including all child nodes, regardless of visibility or CSS. It does not include HTML tags and ignores styling or layout.

Question-3: What is event delegation in the DOM?

Answer : 1. Event Bubbling : When an event (e.g., a click) occurs on a child element, it bubbles up through its parent elements in the DOM hierarchy. This means the parent element can "listen" for events that occur on its children. 2. Event Listener: Instead of attaching an event listener to each child element, you attach a single event listener to a common parent element. The parent element then handles events for all its children. 3. Event Target: Inside the event handler, you use the event.target property to identify the specific child element that triggered the event.
