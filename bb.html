Question-3: What is event delegation in the DOM?

Event delegation is a technique in JavaScript where a single event listener is added to a parent element to handle events on its child elements. This works because of event bubbling.

Question-4: What is event bubbling in the DOM?

Event bubbling is a mechanism where an event starts from the target element and propagates (bubbles) up to its parent elements
